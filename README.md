# Exclusive E-Commerce

Exclusive E-Commerce is a modern storefront project built with Next.js and TypeScript. The application focuses on product discovery, shopping flows, and reusable UI architecture for a polished online shopping experience.

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Material UI
- Radix UI
- NextAuth
- Axios

## Features
- Home, about, and contact pages
- Product browsing and filtered product views
- Search experience
- Shopping cart workflow
- Wishlist management
- Sign up and login pages
- Reusable header, footer, and product-related components
